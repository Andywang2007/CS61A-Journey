#**这里用来记录我学习CS61A这门课的历程，我会包含主要project的源代码，还有做项目，包括学习课程时的学习经验和感悟**

##Project

###1.Hog 在做Hog这个项目的时候其实我的时间跨度非常长，从寒假初就开始利用学雅思晚上的时间做，但是后来就没有继续做下去，直到考完开学考才继续做。1.这个项目给我最大的感受是函数非常多。其实如果有一点计算机基础的话，函数内容的语法实现并不难，我觉得难点在于怎么把根据要求写好的函数串联起来，有点产品经理的意思。后来因为我实在记不住所有的函数，于是把函数名字，功能和参数列表写在记事本里面了，提高了一些效率。

2.因为我先学的c和c++，开始学python在语法上还是有一些不一样，尤其是python中高阶函数的概念，着实让我想了很久。在c/c++中，所有变量的类型都是声明好的，但是在python里面不一样，这也才有了高阶函数，面向对象等等。在学习中，遇到高阶函数时，判断return的结果确实很有挑战性，但是在掌握了python特有的思维方式以后还是蛮有成就感的。

3.我觉得这个项目对于接受国内本科教育的大学生还是很有必要去做一做的，因为它提供了一种不一样的教育方法，去根据要求，引导自己去实现功能。特别依据阅读英语长文然后提炼出功能，我认为对于提升自己计算机水平和英文水平都是很有很大帮助的。

4.现在是2026年4月27日，随着ants项目的写完，我的cs61a学习也就告一段落了。我大概是二月末三月初开始的学习，到现在差不多两个月。后面就先不学了，暂时对我没什么帮助。这两个月的学习给我最大感受是，cs61a这门课真的很强调动手和实践能力，lab，hw，project，这些不只是简单的做题，而且讲知识运用到实际。以后有机会我会向别人推荐这门课。我在学习的过程中很多都以ai辅助，的确是帮我省了不少时间，但还需要多多复习。这个readme就先写到这里，这也是我GitHub第一个项目，我希望后续能在上面上传更多的项目，共勉。
